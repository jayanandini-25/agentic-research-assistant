Under the hood — all 11 phases
It's not just search. It's a whole pipeline.
🧠
Planner Agent
Decomposes your query into 4–6 focused sub-questions ensuring broad coverage and logical structure.
🌐
Parallel Retrieval
Simultaneously queries Tavily, Wikipedia, ArXiv, Semantic Scholar, GitHub, PubMed, and HackerNews.
🔬
Semantic RAG
Chunks all documents, embeds into vector space, and filters by cosine similarity — only relevant bits reach the LLM.
✍️
Summarization Agent
Self-evaluating agent generates 150–250 word summaries per sub-question, retrying if quality score falls below 0.6.
✅
Verification Agent
Check → Judge → Act loop: detects contradictions, weak answers, and source imbalance. Re-retrieves to fix flagged issues.
🔄
LangGraph Orchestration
All agents run as nodes in a LangGraph StateGraph with conditional edges and retry loops for verified quality.
🔭
Deep Research Agent
Recursively expands into dense sub-topics across configurable depth and breadth levels — discovering what standard search misses.
🖼️
Visual Intelligence
BeautifulSoup + Wikipedia REST API + ArXiv figures scraper finds relevant images, embedded inline in the final report.
📝
Writer Agent
Detects query type, plans section structure, writes flowing prose with inline citations [1][2], assigns images to sections, writes abstract + conclusion.
📄
PDF Export
WeasyPrint renders the markdown into a styled A4 PDF with proper typography, page numbers, and citation formatting.
📋
DOCX Export
python-docx generates a Word document with heading styles, superscript citations, embedded images, and a references section.
